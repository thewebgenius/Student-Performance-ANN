# Student Performance Prediction using ANN

## Description
Binary classification model using Artificial Neural Network (ANN)
to predict student performance.

## Technologies
- Python 3.10
- TensorFlow / Keras
- Scikit-learn
- Pandas, NumPy

## Model
- Input layer: 15 features
- Hidden layers: ReLU
- Output layer: Sigmoid
- Loss: Binary Crossentropy
